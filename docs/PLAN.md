# Seed Vault MVP - 구현 계획

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | Seed Vault |
| **컨셉** | Dead Internet 시대, 검증된 인간 지식의 보존소 |
| **대상** | World Build Korea Hackathon 2026 |
| **목표** | World ID Orb 인증 기반 지식 기여/보상 시스템 MVP |
| **팀** | Covalent Collective |

---

## 핵심 가치 제안

1. **진짜 인간의 지식만** - Orb 인증으로 AI 생성 콘텐츠와 봇 스팸 차단
2. **기여에 대한 보상** - 지식이 인용될 때마다 WLD 토큰 보상
3. **투명한 출처** - 답변의 근거가 되는 지식 노드를 그래프로 시각화

## 화면 구성

| 화면 | 경로 | 설명 |
|------|------|------|
| 랜딩 / 저널링 홈 | `/` | World ID 로그인 → 음성 저널링 + Vault 기여 |
| Vault 탐색 | `/explore` | 3D 카루셀로 전문가 봇 탐색 |
| 봇 상세 | `/explore/[botId]` | 지식 그래프 + 질문/답변 + 기여 영수증 |
| 보상 대시보드 | `/rewards` | 파워 레벨, WLD 보상, 거래 로그 |

## MVP 범위 (In Scope)

- World ID Orb 인증 (MiniKit)
- 전문가 봇 7개 (World Coin 전문가, 서울 로컬 가이드, 산부인과 전문의, 한식 레시피 마스터, 스타트업 멘토, 크립토 전문가, K-POP 인사이더)
- 음성 저널링 인터페이스 (데모용 시뮬레이션)
- 글래스모피즘 디자인 시스템 (커스텀)
- 3D 카루셀 봇 탐색
- 지식 그래프 시각화 (2D)
- 상세 기여 영수증 (비례 분배)
- 보상 UI (실제 트랜잭션 없음)

## MVP 범위 밖 (Out of Scope)

- 실제 WLD 토큰 전송
- 백엔드 서버 / 데이터베이스
- AI 답변 생성 (Mock으로 대체)
- 사용자 간 소셜 기능

## 성공 기준

### 기술적 성공 (MVP)
- [x] World App 내 Orb 인증 완료율 > 90% (Mock 모드)
- [x] 지식 기여 flow 완료율 > 80%
- [x] 그래프 렌더링 2초 이내
- [x] 데모 시나리오 3분 이내 완료

### 제품 성공 (해커톤 기간)
- [ ] 테스트 사용자 10명 이상 확보
- [ ] 기여된 지식 노드 50개 이상
- [ ] 사용자 NPS > 7 (10점 만점)

### 비즈니스 성공 (Post-Hackathon)
- [ ] 월간 활성 기여자 100명 (3개월 내)
- [ ] 지식 노드 1,000개 (6개월 내)
- [ ] 일평균 질문 500회 (6개월 내)

### 핵심 성과 지표 (KPIs)

| 지표 | 정의 | 목표 |
|------|------|------|
| **기여 전환율** | 방문자 중 기여 완료 비율 | > 5% |
| **재방문율** | 7일 내 재방문 비율 | > 30% |
| **인용 밀도** | 노드당 평균 인용 횟수 | > 3회 |
| **기여자 활성도** | 기여자당 월평균 노드 수 | > 2개 |

## 리스크 & 대응

### 기술 리스크

| 리스크 | 가능성 | 영향 | 대응 |
|--------|--------|------|------|
| World App 외부 실행 | 높음 | 중간 | Mock 모드 폴백 + 안내 메시지 |
| 그래프 렌더링 성능 | 중간 | 중간 | 노드 30개 이하 제한, 2D만 사용 |
| MiniKit 인증 실패 | 낮음 | 높음 | Staging 모드 테스트, 에러 핸들링 |
| Demo day 네트워크 불안정 | 중간 | 높음 | 사전 녹화 백업 + Mock 모드 |
| Vercel 배포 실패 | 낮음 | 높음 | 로컬 ngrok 백업 준비 |

### 비즈니스 리스크

| 리스크 | 가능성 | 영향 | 대응 |

## Post-MVP 백엔드 마일스톤

| Milestone | Tasks | Priority |
|-----------|-------|----------|
| **API Server** | Fastify 설정, World ID 검증 엔드포인트 | P0 |
| **Database** | PostgreSQL 스키마, Prisma 마이그레이션 | P0 |
| **Auth Flow** | 서버사이드 proof 검증, JWT 발급 | P0 |
| **Contribution API** | 노드 CRUD, 중복 감지 | P1 |
| **Query Engine** | 키워드 매칭, 인용 계산 | P1 |
| **Rewards Ledger** | 보상 추적, claim flow | P2 |
| **WLD Integration** | 스마트 컨트랙트 연동, 토큰 전송 | P2 |
