# Seed Vault - Pitch Overview

## English Summary

**Seed Vault** is a human-verified knowledge repository powered by World ID Orb authentication. In an era where AI-generated content dominates the internet, Seed Vault ensures that only verified humans can contribute knowledge, creating a trusted source of authentic human experiences.

---

## 🎯 문제 정의

### Dead Internet이 현실이 되다

| 연도 | AI 생성 콘텐츠 비율 |
|------|---------------------|
| 2024 | 50% |
| 2025 | 70% |
| 2026 | 90% (예상) |

**출처**: Europol 2024 보고서 "Facing Reality" - 2026년까지 온라인 콘텐츠의 90%가 AI에 의해 생성될 수 있다고 예측

### 추가 데이터 포인트
- NewsGuard 2024: 1,000개 이상의 AI 생성 뉴스 사이트 식별
- AI 생성 리뷰, 댓글, 소셜 미디어 포스트 급증
- 검색 엔진 결과의 신뢰도 하락

### 사용자 Pain Point
- "검색 결과가 다 광고 아니면 AI가 쓴 것 같아요"
- "진짜 경험담은 어디서 찾죠?"
- "누가 쓴 건지 알 수 없어서 불안해요"

### 문제 검증 (사용자 인터뷰 n=15)
- 87%가 "온라인 정보 신뢰도 하락" 경험
- 73%가 "실제 경험담 찾기 어려움" 응답
- 60%가 "신뢰할 수 있는 출처에 비용 지불 의향"

## 🔄 작동 방식

### 기여자 Flow
```
Orb 인증 → 음성 저널링 → AI 키워드 추출 → Vault 선택 → 기여 → 보상
```

### 검색자 Flow
```
Vault 탐색 → 질문 → 지식 그래프 검색 → 답변 + 기여 영수증
```
